El hidróxido de litio reacciona con el dióxido de carbono gaseoso para formar carbonato de litio (\ce{Li2CO3}) sólido y agua líquida (\ce{H2O}). ¿Cuántos gramos de dióxido de carbono puede absorber 1,00 g de hidróxido de litio?
